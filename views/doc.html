Getting Started

If you are the first time

Open a terminal window and make an API GET request with a URL as query parameter for a resource.
curl http://localhost:5500/ytt/api?url=https://youtu.be/MBdVXkSdhwU
If the request is valid, you will see a successful message as follows.
the highlighted value at the end of URL is the ID of the document, which means, a document with the URL input exists in our DB.
Simply copy it.
Found. Redirecting to /ytt/api/61aa5f048eb33d6305a522bb
Now, make an actual GET request with the ID.
curl http://localhost:5500/ytt/api/61aa5f048eb33d6305a522bb
The response you get. 
        {
          "_id": "61aa5f048eb33d6305a522bb",
          "title": "BTS (방탄소년단) 'DNA' Official MV",
          "url": "https://youtu.be/MBdVXkSdhwU",
          "available languages in captionTracks": [
              "0: .en",
              "1: a.ko"
          ]
        }
