Introduction

YTT_API is a RESTful API that provides text reproduction service from subtitles, CC, or script registered in Youtube videos. Take a look at our API documentation and our web app that interacts with the API.

YTT API is where you can get a fully-mgerged subtitles, CC, or script (hereinafter referred to as "script") of a Youtube video in one go without having to play any videos from Youtube. You can get a script info of a video in JSON, a specific script of a video by a language, languages available, and more.
The API is web-based and the supported language for reuqest & response is javascript. Any further language supports will be updated via this documentation page.
Getting Started and Resources will guildes you on how to use the API with HTTP requests. Welcome to YTT API !

Getting Started

curl

Open a terminal window and make an API GET request with a URL as query parameter for a resource.
curl http://localhost:5500/ytt/api?url=https://youtu.be/MBdVXkSdhwU
If the request is valid, you will see a successful message as follows.
the highlighted value at the end of URL is the ID of the document, which means, a document with the URL input exists in our DB.
Simply copy it.
Found. Redirecting to /ytt/api/61aa5f048eb33d6305a522bb
Now, make an actual GET request with the ID.
curl http://localhost:5500/ytt/api/61aa5f048eb33d6305a522bb
The response you get. 
        {
          "_id": "61aa5f048eb33d6305a522bb",
          "title": "BTS (방탄소년단) 'DNA' Official MV",
          "url": "https://youtu.be/MBdVXkSdhwU",
          "available languages in captionTracks": [
              "0: .en",
              "1: a.ko"
          ]
        }

Resources

POST /ytt/api?url=

Create and save a new document to DB by providing a YOUTUBE video URL as a query parameter.
(The actual HTTP verb operation used is GET, which functions as a POST in this endpoint)
Option 1) send a POST request and GET the document
example request
http://localhost:5500/ytt/api?url=https://youtu.be/MBdVXkSdhwU
example response
POST to the collection successfully
A successful response will automatically redirect to the endpoint GET /ytt/api/:id
Click here to check the response
Option 2) send a POST request and GET the document with full details
example request
http://localhost:5500/ytt/api?url=https://youtu.be/MBdVXkSdhwU&full=true
example response
POST to the collection successfully
A successful response will automatically redirect to the endpoint GET /ytt/api/:id?full=true
Click here to check the response
